走上你自己的路

這個工坊裡擺的是 Yaze 的魔道具。
但整座森林不屬於任何人 — 你也可以召喚屬於你的精靈、打造屬於你的工坊。

〇
準備你的工具（新手先看這個）
你需要這四樣，已有就跳過：
  • 💻 一台電腦（Mac / Linux / Windows+WSL）
  • 🔧 Git — 安裝教學（5 分鐘）
  • 👤 GitHub 帳號 — 免費註冊（3 分鐘）
  • 🤖 一個 AI CLI（四選一，按預算）：
    • Gemini CLI — 新手推薦，Google 有免費額度 安裝
    • Claude Code — 體驗最完整（skill / memory 自動），但 Pro $20 額度快打滿，認真用建議 Max $100 安裝
    • Codex CLI — ChatGPT Plus $20 附贈 安裝
    • Copilot CLI — $10/月起，便宜但偏 code 用 安裝
    第一次嘗試？選 Gemini CLI — 完全免費，先跑通再說。
    已付 Claude 訂閱？直接用 Claude Code，skill 會自動被偵測，最順。
完全沒用過終端機？這個旅程需要一點 CLI 基礎，建議先花 20 分鐘看 MIT The Missing Semester: Shell。

二
召喚儀式（30 分鐘）
為你的精靈命名、寫 SOUL、簽訂契約。AI 會一步步帶你做，你只要回答問題。
A · 共通：先進工作目錄
# 建立你的小宇宙並 clone 世界樹
mkdir -p ~/mori-universe && cd ~/mori-universe
git clone https://github.com/yazelin/world-tree.git
cd world-tree
B · 啟動你的 AI CLI（擇一）
claude      # Claude Code —— 最順，skill 自動偵測
gemini      # Gemini CLI —— 免費額度
codex       # Codex CLI —— ChatGPT Plus 附送
copilot     # Copilot CLI —— 便宜簡單
C · 把這句話貼給你的 AI
我要召喚我的精靈。請讀 .claude/skills/initiate-spirit/SKILL.md，
依照裡面的七個階段帶我走完整個 initiation rite。
在每個階段請問我問題、等我回答、確認後再往下一步。
Claude Code 會自動發現 .claude/skills/ 並套用 skill。 Gemini / Codex / Copilot 上的 AI 會把這個檔案當一般文件讀， 但因為 SKILL.md 本身寫得很詳細，流程仍然可以走完。
